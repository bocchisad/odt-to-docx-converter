# Конвертер ODT в DOCX

Инструмент для конвертации документов, работающий на стороне клиента. Преобразует файлы OpenDocument Text (.odt) в формат Microsoft Word (.docx). Вся обработка происходит в браузере без загрузки файлов на сервер.

## Обзор

Это автономное HTML-приложение, которое предоставляет веб-интерфейс для конвертации документов ODT в формат DOCX. Конвертер извлекает текстовое содержимое из файлов ODT (которые являются ZIP-архивами) и генерирует правильно отформатированные документы DOCX с использованием библиотеки docx.

## Функциональность

- Конвертация на стороне клиента без загрузки файлов на серверы
- Интерфейс с поддержкой перетаскивания (drag-and-drop)
- Валидация файла и отображение размера
- Обратная связь о статусе конвертации в реальном времени
- Обработка ошибок с детальными сообщениями об ошибках
- Адаптивный дизайн для десктопа и мобильных устройств
- Состояние загрузки с анимацией спиннера
- Прямое скачивание преобразованных файлов
- Работа с HTML5 File API и XMLHttpRequest

## Технологический стек

- HTML5
- CSS3 с CSS Grid и Flexbox
- Vanilla JavaScript (ES6+)
- JSZip 3.10.1 - для извлечения содержимого файлов ODT
- Docx 8.5.0 - для создания документов DOCX

## Как это работает

1. Пользователь выбирает или перетаскивает файл ODT
2. Файл проверяется на расширение .odt
3. При конвертации:
   - Файл читается как ArrayBuffer
   - JSZip извлекает содержимое из ZIP структуры
   - content.xml анализируется для извлечения текстовых абзацев
   - XML обрабатывается DOMParser для извлечения текстовых узлов
   - Библиотека Docx создает новый Document с извлеченными абзацами
   - Document упаковывается как Blob и предлагается для скачивания

## Структура кода

### HTML компоненты
- Область для загрузки файлов
- Элемент файлового ввода (скрыт)
- Отображение информации о файле с названием и размером
- Кнопка конвертации
- Область статуса/хода выполнения с динамическими сообщениями
- Генерация ссылки для скачивания

### CSS классы

| Класс | Назначение |
|-------|-----------|
| converter-card | Главный контейнер |
| dropzone | Область для загрузки файлов |
| file-input-label | Кнопка выбора файла |
| convert-btn | Кнопка запуска конвертации |
| status-area | Область отображения сообщений |
| spinner | Анимация загрузки |

### JavaScript функции

**Работа с файлами:**
- `handleFile(file)` - Валидирует и обрабатывает выбранный файл
- `resetFileSelection()` - Очищает текущий файл и состояние UI
- `updateFileInfoUI(file)` - Отображает название и размер файла

**Конвертация:**
- `convertOdtToDocx(file)` - Основная функция конвертации
- `extractTextFromXml(xmlString)` - Анализирует содержимое ODT XML
- `startConversion(file)` - Инициирует асинхронный процесс конвертации

**Обратная связь с пользователем:**
- `showStatus(message, isError, downloadUrl, fileName)` - Отображает сообщения статуса
- `showLoadingStatus()` - Показывает процесс конвертации
- `clearStatus()` - Удаляет сообщения статуса
- `escapeHtml(str)` - Санитизирует HTML содержимое

**Обработчики событий:**
- События изменения файлового ввода
- События перетаскивания (dragover, dragleave, drop)
- События клика на область загрузки
- Клик кнопки конвертации

Протестировано на: Chrome 80+, Firefox 75+, Safari 13+, Edge 80+

## Ограничения

- Извлекает только простой текст из файлов ODT
- Расширенное форматирование (жирный, курсив, цвета) не сохраняется
- Таблицы и изображения не преобразуются
- Сложные структуры ODT могут привести к выводу только текста
- Большие файлы (>10MB) могут испытывать замедление производительности

## Использование

1. Откройте HTML файл в веб-браузере
2. Выберите или перетащите файл ODT в зону загрузки
3. Нажмите "Конвертировать в DOCX"
4. Скачайте полученный файл DOCX

## Валидация файла

- Принимает только файлы с расширением .odt
- Проверяет наличие content.xml в ZIP структуре
- Валидирует анализ XML перед конвертацией

## Обработка ошибок

Приложение предоставляет специфичные сообщения об ошибках для:
- Неверного формата файла
- Отсутствующих или поврежденных content.xml
- Отсутствующих требуемых библиотек (JSZip, Docx)
- Ошибок анализа XML
- Проблем сети, препятствующих загрузке библиотек

- Большие файлы могут вызвать временную неотзывчивость UI
- Скачивания обрабатываются через создание Blob и URL.createObjectURL()
