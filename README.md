# Универсальный конвертер документов и изображений

Многофункциональный инструмент для конвертации документов и изображений, работающий полностью на стороне клиента. Поддерживает конвертацию множества форматов файлов без загрузки на серверы.

## Обзор

Это универсальное HTML5 приложение, которое предоставляет веб-интерфейс для конвертации:
- Документов офисного формата (ODT, DOCX, TXT, CSV, XLSX, Markdown, HTML)
- Изображений (PNG, JPG, WEBP)
- В различные целевые форматы (PDF, DOCX, CSV, XLSX, PNG, JPG, HTML)

Все преобразования происходят полностью в браузере пользователя без отправки данных на сервер.

## Функциональность

- Конвертация на стороне клиента без загрузки файлов на серверы
- Поддержка 11+ форматов конвертации
- Интерфейс с поддержкой перетаскивания (drag-and-drop)
- Валидация файла и отображение размера
- Обратная связь о статусе конвертации в реальном времени
- Обработка ошибок с детальными сообщениями об ошибках
- Адаптивный дизайн для десктопа и мобильных устройств
- Состояние загрузки с анимацией спиннера
- Прямое скачивание преобразованных файлов
- Работа с HTML5 File API и XMLHttpRequest

## Поддерживаемые конвертации

### Документы Office
- ODT → DOCX (OpenDocument в Word)
- DOCX → PDF (Word в PDF)
- TXT → DOCX (Текст в Word)

### Электронные таблицы
- CSV → XLSX (CSV в Excel)
- XLSX → CSV (Excel в CSV)

### Изображения
- PNG → JPG (без потерь в JPG)
- JPG → PNG (JPEG в PNG)
- WEBP → PNG (WebP в PNG)
- WEBP → JPG (WebP в JPEG)

### Веб-форматы
- Markdown → HTML (Markdown в HTML)
- HTML → PDF (HTML в PDF)

## Как это работает

1. Пользователь выбирает тип конвертации из выпадающего списка
2. Пользователь выбирает или перетаскивает файл
3. Файл валидируется на совместимость с выбранным преобразованием
4. При конвертации:
   - Файл читается как ArrayBuffer или Text
   - В зависимости от типа конвертации используются соответствующие библиотеки
   - Для документов - извлечение контента и преобразование в целевой формат
   - Для изображений - загрузка через Canvas API и преобразование через toBlob()
   - Для таблиц - парсинг и генерация новой структуры
   - Результат упаковывается как Blob и предлагается для скачивания

### JavaScript функции

### JavaScript функции

**Выбор типа конвертации:**
- `conversionRules` - объект с описанием всех поддерживаемых конвертаций
- `validateFileFormat(file)` - проверка совместимости файла с выбранным типом

**Конвертация документов:**
- `convertOdtToDocx(file)` - конвертация ODT в DOCX с улучшенной обработкой
- `convertDocxToPdf(file)` - конвертация DOCX в PDF
- `convertTxtToDocx(file)` - преобразование простого текста в DOCX
- `convertMarkdownToHtml(file)` - преобразование Markdown в HTML

**Конвертация таблиц:**
- `convertCsvToXlsx(file)` - преобразование CSV в Excel
- `convertXlsxToCsv(file)` - преобразование Excel в CSV

**Конвертация изображений:**
- `convertImageFormat(file, targetFormat)` - универсальная функция для конвертации изображений через Canvas API

**Работа с файлами:**
- `handleFile(file)` - валидирует и обрабатывает выбранный файл
- `resetFileSelection()` - очищает текущий файл и состояние UI
- `updateFileInfoUI(file)` - отображает название и размер файла

**Конвертация:**
- `startConversion(file)` - инициирует процесс конвертации по типу
- `escapeHtml(str)` - санитизирует HTML содержимое

**Обратная связь с пользователем:**
- `showStatus(message, isError, downloadUrl, fileName)` - отображает сообщения статуса
- `showLoadingStatus()` - показывает процесс конвертации
- `clearStatus()` - удаляет сообщения статуса

**Обработчики событий:**
- События изменения файлового ввода
- События перетаскивания (dragover, dragleave, drop)
- События клика на область загрузки
- Клик кнопки конвертации
- Изменение выбора типа конвертации

## Ограничения и особенности

### Документы
- ODT в DOCX: извлекает текст с улучшенной обработкой параграфов
- Сложное форматирование (таблицы, изображения внутри документов) требует серверной обработки
- DOCX в PDF: требует полной поддержки на сервере для полной функциональности

### Изображения
- Конвертация изображений работает через Canvas API
- JPG конвертируется с качеством 85% для оптимального размера
- Большие изображения (>10MB) могут вызвать задержку

### Таблицы
- CSV парсится по запятым (не поддерживает сложные разделители)
- XLSX конвертирует только первый лист

### Общее
- Большие файлы (>50MB) могут запросить больше памяти браузера
- Некоторые конвертации (DOCX→PDF, HTML→PDF) работают на базовом уровне
- Обработка происходит в основном потоке браузера

## Использование

1. Откройте index.html в веб-браузере
2. Выберите нужный тип конвертации из выпадающего списка
3. Перетащите файл в зону загрузки или нажмите "Выбрать файл"
4. Нажмите кнопку "Конвертировать"
5. Скачайте готовый файл

## Валидация файла

- Проверяет расширение файла перед конвертацией
- Валидирует совместимость файла с выбранным типом конвертации
- Отображает понятные сообщения об ошибках при несовместимости

## Обработка ошибок

Приложение предоставляет специфичные сообщения об ошибках для:
- Неверного формата файла
- Несовместимости файла с выбранным типом конвертации
- Отсутствующих требуемых частей файла (например, content.xml в ODT)
- Ошибок парсинга и обработки данных
- Проблем при работе с библиотеками конвертации
